#Developing tools

## PHP Unit Test

Atoum is the Unit Test, the command to test idrectly is:

```
./vendor/bin/atoum -f tests/units/HelloWord.php
./vendor/bin/atoum -f tests/units/gd-mylist.php
```

It means call the tools and then the testing file. More info [here](https://atoum.readthedocs.io/en/latest/index.html)
